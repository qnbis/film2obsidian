# Obsidian Film Saver + HDRezka Player 🍿

Комплексное решение для интеграции фильмов и сериалов с **HDrezka (rezka.ag)** в вашу базу знаний **Obsidian**. 

Состоит из двух компонентов:
1. **Chrome Расширение (Obsidian Film Saver)**: Позволяет в один клик сохранить страницу фильма со всеми метаданными (название, год, жанры, рейтинги) и обложкой прямо в ваш Vault.
2. **Obsidian Плагин (HDRezka Player)**: Позволяет **смотреть** сохраненные фильмы прямо внутри заметок Obsidian в красивом встроенном плеере (поддержка HLS, выбор качества, озвучки, субтитров и скорости).

---

## 🌟 Возможности

### Расширение для браузера
- 📥 Автоматический сбор данных о фильме/сериале (название, год, жанры, страны, режиссеры, актеры, рейтинги IMDb и Кинопоиск).
- 🎨 Красивый всплывающий Glassmorphism интерфейс для добавления личной оценки и просмотра последних сохраненных фильмов.
- 🖼 Скачивание постера (обложки) в нужном формате.
- 🚀 Прямая бесшовная загрузка `.md` заметки и файла постера в ваш Vault по локальной сети через Local REST API.

### Плагин для Obsidian (HDRezka Player)
- 🍿 **Встроенный плеер**: Воспроизведение видео прямо в Obsidian по ссылке с rezka.ag.
- 🎭 **Смена озвучки**: Переключение перевода/озвучки прямо в плеере без перезагрузки страницы.
- 📺 **Выбор качества**: Динамическое переключение качества (от 360p до 1080p).
- 🚀 **Контроль скорости**: Ускорение или замедление воспроизведения.
- 💅 **Премиальный дизайн**: Кастомный темный интерфейс с фиолетовыми акцентами, размытием и удобным прогресс-баром.
- 🛡 **Обход блокировок CORS**: Встроенный локальный прокси-сервер обеспечивает бесперебойную загрузку видео даже при строгих ограничениях сети.

## ⚙️ Установка и настройка

### 1. Настройка Obsidian
1. Откройте Obsidian и зайдите в **Настройки** -> **Community plugins**.
2. Выключите "Restricted mode".
3. Нажмите кнопку **Browse** и найдите плагин `Local REST API`. Установите и включите его.
4. В настройках Local REST API:
   - **Включите:** `Enable Non-Encrypted (HTTP) Server`.
   - **Скопируйте `API Key`** (копируйте только сам ключ, без слова `Bearer`).

### 2. Установка браузерного расширения
1. Скачайте этот репозиторий.
2. В Chrome перейдите по адресу: `chrome://extensions/`.
3. Включите **«Режим разработчика»** (Developer mode).
4. Нажмите **«Загрузить распакованное расширение»** и выберите корневую папку репозитория.
5. Нажмите по иконке расширения правой кнопкой мыши -> **«Параметры»**.
6. Вставьте ваш `API Key` из Obsidian и сохраните.

### 3. Установка плагина HDRezka Player в Obsidian
1. В вашем Obsidian Vault перейдите в папку `.obsidian/plugins/`.
2. Создайте папку `obsidian-hdrezka-player`.
3. Скопируйте файлы из папки `obsidian-player` этого репозитория (`main.js`, `manifest.json`, `hls.min.js`) в созданную папку.
4. Перезапустите Obsidian (или обновите список плагинов) и включите плагин **HDRezka Player**.

## 💻 Использование

1. Зайдите на любую страницу фильма на сайте HDrezka (например, `rezka.ag/films/...`).
2. Нажмите на иконку расширения в браузере и кликните "Сохранить в Obsidian" 🎬.
3. Расширение создаст заметку в вашем Vault.
4. В самой заметке вы можете добавить блок:
   \`\`\`hdrezka
   https://rezka.ag/films/thriller/87731-dzhek-rayan-prizrachnaya-voyna-2026.html
   \`\`\`
   Плагин превратит эту ссылку в полнофункциональный видеоплеер!

## 📁 Структура папок в Obsidian
По умолчанию расширение сохраняет фильмы по пути:
- `Фильмы и Сериалы/Фильмы/Имя Фильма (Год).md`
- Постер сохраняется в `Фильмы и Сериалы/Фильмы/Постеры/...`

*(Базовую папку можно легко поменять в настройках расширения)*
